ex.Camera

A base implementation of a camera. This class is meant to be extended.

Constructor

new ex.Camera(engine)
Parameters:
Name Type Description
engine Engine

Reference to the current engine

Methods

getFocus( ): Point

Returns the focal point of the camera

Number 
getZoom( ): the current zoom scale

gets the current zoom scale

setActorToFollow(actor)

Sets the {{#crossLink Actor}}{{//crossLink}} to follow with the camera

Parameters:
Name Type Description
actor Actor

The actor to follow

 

setFocus(x, y)

Sets the focal point of the camera. This value can only be set if there is no actor to be followed.

Parameters:
Name Type Description
x Number

The x coordinate of the focal point
y Number

The y coordinate of the focal point

 

shake(magnitudeX, magnitudeY, duration)

Sets the camera to shake at the specified magnitudes for the specified duration

Parameters:
Name Type Description
magnitudeX Number

the x magnitude of the shake
magnitudeY Number

the y magnitude of the shake
duration Number

the duration of the shake

 

update(delta)

Applies the relevant transformations to the game canvas to "move" or apply effects to the Camera

Parameters:
Name Type Description
delta Number

The number of milliseconds since the last update

 

zoom(scale, [duration])

Zooms the camera in or out by the specified scale over the specified duration. If no duration is specified, it will zoom by a set amount until the scale is reached.

Parameters:
Name Type Description
scale Number

the scale of the zoom
[duration] Number

the duration of the zoom
