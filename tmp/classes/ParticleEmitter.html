ex.ParticleEmitter

Using a particle emitter is a great way to create interesting effects in your game, like smoke, fire, water, explosions, etc. Particle Emitters extend Actor allowing you to use all of the features that come with Actor

Constructor

new ex.ParticleEmitter([x=0], [y=0], [width=0], [height=0])
Parameters:
Name Type Description
[x=0] Number

The x position of the emitter
[y=0] Number

The y position of the emitter
[width=0] Number

The width of the emitter
[height=0] Number

The height of the emitter

Methods

emit(particleCount)

Causes the emitter to emit particles

Parameters:
Name Type Description
particleCount Number

Number of particles to emit right now

Properties

[acceleration=new Vector(0,0)]: Vector

Gets or sets the acceleration vector for all particles

[beginColor=Color.White]: Color

Gets or sets the beginning color of all particles

[emitRate=1]: Number

Gets or sets the emission rate for particles (particles/sec)

[emitterType=EmitterType.Rectangle]: EmitterType

Gets or sets the emitter type for the particle emitter

[endColor=Color.White]: Color

Gets or sets the ending color of all particles

[fade=false]: Boolean

Gets or sets the fade flag which causes particles to gradually fade out over the course of their life.

[focus=null]: Vector

Gets or sets the optional focus where all particles should accelerate towards

[focusAccel=1]: Number

Gets or sets the acceleration for focusing particles if a focus has been specified

[maxAngle=0]: Number

Gets or sets the maximum angle in radians

[maxSize=5]: Number

Gets or sets the maximum size of all particles

[maxVel=0]: Number

Gets or sets the maximum partical velocity

[minAngle=0]: Number

Gets or sets the minimum angle in radians

[minSize=5]: Number

Gets or sets the minimum size of all particles

[minVel=0]: Number

Gets or sets the minimum partical velocity

[opacity=1.0]: Number

Gets or sets the opacity of each particle from 0 to 1.0

[particleLife=2000]: Number

Gets or sets the life of each particle in milliseconds

[particleSprite=null]: Sprite

Gets or sets the sprite that a particle should use

[radius=0]: Number

Gets or sets the emitter radius, only takes effect when the emitterType is Circle

isEmitting: Boolean

Gets or sets the isEmitting flag

particles: Util.Collection<Particle>

Gets or sets the backing particle collection
