Polygon

Extends: IDrawable

Creates a closed polygon drawing given a list a of points. Polygons should be used sparingly as there is a performance impact for using them.

Constructor

new Polygon(points)
Parameters:
Name Type Description
points Point[]

The points to use to build the polygon in order
