Engine

The 'Engine' is the main driver for a game. It is responsible for starting/stopping the game, maintaining state, transmitting events, loading resources, and managing the scene.

Constructor

new Engine([width], [height], [canvasElementId], [displayMode])
Parameters:
Name Type Description
[width] Number

The width in pixels of the Excalibur game viewport
[height] Number

The height in pixels of the Excalibur game viewport
[canvasElementId] String

If this is not specified, then a new canvas will be created and inserted into the body.
[displayMode] DisplayMode

If this is not specified, then it will fall back to fixed if a height and width are specified, else the display mode will be FullScreen.

Methods

addChild(actor)

Adds an actor to the current scene of the game. This is synonymous to calling engine.currentScene.addChild(actor : Actor).

Actors can only be drawn if they are a member of a scene, and only the 'currentScene' may be drawn or updated.

Parameters:
Name Type Description
actor Actor

The actor to add to the current scene

 

addScene(name, scene)

Adds a scene to the engine, think of scenes in excalibur as you would scenes in a play.

Parameters:
Name Type Description
name String

The name of the scene, must be unique
scene Scene

The scene to add to the engine

 

addTimer(timer)

Adds an excalibur timer to the current scene.

Parameters:
Name Type Description
timer Timer

The timer to add to the current scene.

 

getAntialiasing( ): boolean

Return the current smoothing status of the canvas

getHeight( ): number The height of the drawing surface in pixels.

Returns the height of the engines drawing surface in pixels.

getWidth( ): number The width of the drawing surface in pixels.

Returns the width of the engines drawing surface in pixels.

goToScene(name)

Changes the currently updating and drawing scene to a different, named scene.

Parameters:
Name Type Description
name String

The name of the scene to trasition to.

Test wether a key is down

 

isKeyPressed(key): boolean

Tests if a certain key is pressed.

Parameters:
Name Type Description
key InputKey

Test wether a key is pressed

 

isKeyUp(key): boolean

Tests if a certain key is up.

Parameters:
Name Type Description
key InputKey

Test wether a key is up

 

load(loader)

Another option available to you to load resources into the game. Immediately after calling this the game will pause and the loading screen will appear.

Parameters:
Name Type Description
loader ILoadable

Some loadable such as a Loader collection, Sound, or Texture.

 

playAnimation(animation, x, y)

Plays a sprite animation on the screen at the specified x and y (in game coordinates, not screen pixels). These animations play independent of actors, and will be cleaned up internally as soon as they are complete. Note animations that loop will never be cleaned up.

Parameters:
Name Type Description
animation Animation

Animation to play
x Number

x game coordinate to play the animation
y Number

y game coordinate to play the animation

 

removeChild(actor)

Removes an actor from the currentScene of the game. This is synonymous to calling engine.currentScene.removeChild(actor : Actor). Actors that are removed from a scene will no longer be drawn or updated.

Parameters:
Name Type Description
actor Actor

The actor to remove from the current scene.

 

removeTimer(timer)

Removes an excalibur timer from the current scene.

Parameters:
Name Type Description
timer Timer

The timer to remove to the current scene.

 

screenToWorldCoordinates(point)

Transforms the current x, y from screen coordinates to world coordinates

Parameters:
Name Type Description
point Point

screen coordinate to convert

 

setAntialiasing(isSmooth)

If supported by the browser, this will set the antialiasing flag on the canvas. Set this to false if you want a 'jagged' pixel art look to your image resources.

Parameters:
Name Type Description
isSmooth Boolean

Set smoothing to true or false

 

setLoadingDrawFunction(fcn)

Sets the loading screen draw function if you want to customize the draw

Parameters:
Name Type Description
fcn Ctx: CanvasRenderingContext2D, loaded: number, total: number) => void

Callback to draw the loading screen which is passed a rendering context, the number of bytes loaded, and the total number of bytes to load.

 

start([loader=undefined]): Promise

Starts the internal game loop for Excalibur after loading any provided assets.

Parameters:
Name Type Description
[loader=undefined] ILoadable

Optional resources to load before starting the mainloop. Some loadable such as a Loader collection, Sound, or Texture.

 

stop( )

Stops Excalibur's mainloop, useful for pausing the game.

worldToScreenCoordinates(point)

Transforms a world coordinate, to a screen coordinate

Parameters:
Name Type Description
point Point

world coordinate to convert

Properties

[backgroundColor=new Color(0, 0, 100)]: Color

Sets the background color for the engine.

[displayMode=FullScreen]: DisplayMode

Indicates the current DisplayMode of the engine.

[isDebug=false]: Boolean

Indicates whether the engine should draw with debug information

canvas: HTMLCanvasElement

Direct access to the engine's canvas element

canvasElementId: String

Direct access to the canvas element id, if an id exists

ctx: CanvasRenderingContext2D

Direct access to the engine's 2D rendering context

height: Number

The height of the game canvas in pixels

isFullscreen: Boolean

Indicates whether the engine is set to fullscreen or not

rootScene: Scene

The default scene of the game, use goToScene to transition to different scenes.

width: Number

The width of the game canvas in pixels
