Effects.Opacity

Extends: ISpriteEffect

Applies the "Opacity" effect to a sprite, setting the alpha of all pixels to a given value.

Constructor

new Effects.Opacity(opacity)
Parameters:
Name Type Description
opacity Number

The new opacity of the sprite from 0-255
