SpriteFont

Extends: SpriteSheet

SpriteFonts are a used in conjunction with a to specify a particular bitmap as a font.

Constructor

new SpriteFont(image, alphabet, caseInsensitve, columns, rows, spWdith, spHeight)
Parameters:
Name Type Description
image Texture

The backing image texture to build the SpriteFont
alphabet String

A string representing all the charaters in the image, in row major order.
caseInsensitve Boolean

Indicate whether this font takes case into account
columns Number

The number of columns of characters in the image
rows Number

The number of rows of characters in the image
spWdith Number

The width of each character in pixels
spHeight Number

The height of each character in pixels

Methods

draw(ctx, x, y, text)

Draw a particalur string to rendering context using this font

Parameters:
Name Type Description
ctx CanvasRenderingContext2D

The 2D rendering context
x Number

The x position on the canvas to place the text
y Number

The y position on the canvas to place the text
text String

The text to write to the screen

getAnimationByIndices(engine, indices, speed): Animation
Defined in SpriteSheet:

Create an animation from the this SpriteSheet by listing out the sprite indices. Sprites are organized in row major order in the SpriteSheet.

Parameters:
Name Type Description
engine Engine

Reference to the current game Engine
indices Number

An array of sprite indices to use in the animation
speed Number

The number in milliseconds to display each frame in the animation
