Texture

The Texture object allows games built in Excalibur to load image resources. It is generally recommended to preload images using the "Texture" object.

Usage

// Load image into Excalibur
var game = new ex.Engine();
var loader = new ex.Loader();
var texture = new ex.Texture("someimage.png");
loader.addResource(texture);

game.start(loader);

Constructor

new(path: string)

The Texture constructor takes a path to an audio file.

Properties

texture.image: HTMLImageElement

Gets the internal HTMLImageElement once the image has been loaded.

Methods

load(): ex.Promise

Begins loading the image and returns a promise that will be resolved when loading is complete.
